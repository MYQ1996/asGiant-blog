> 使用 const 我们可以声明一个常量，一旦声明之后，就不可以更改。

## 常量不能重新赋值

```javascript
const MY_NAME = 'zfpx';
MY_NAME = 'zfpx2';//Assignment to constant variable
```

## 变量可以改变

> 注意const限制的是不能给变量重新赋值，而变量的值本身是可以改变的,下面的操作是可以的

```javascript
const names = ['zfpx1'];
names.push('zfpx2');
console.log(names);
```

## 不同的块级作用域可以多次定义

```javascript
const A = "0";
{
    const A = "A";
    console.log(A)
}
{
    const A = "B";
    console.log(A)
}
console.log(A)

// 结果 A B 0
```
