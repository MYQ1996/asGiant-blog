7.Math的常用方法

2019-04-10

次访问

数学函数

但是他是对象数据类型的

  • ‘object’
  • Math对象中给我们提供了很多常用操作数字的方法
  • console.dir(Math) // 查看所有方法

abs

Math.abs 取绝对值

cell / floor

cell: 向上取整
floor: 向下取整

round

round: 四舍五入

random

random: 获取一个[0,1]之间的一个随机小数

max/minx

max 获取一组值中的最大值
minx 获取一组值中的最小值

PI

Math.PI 获取圆周率

pow / sqrt

pow 获取一个值的多少幂
sqrt 获取一个值的开平方
