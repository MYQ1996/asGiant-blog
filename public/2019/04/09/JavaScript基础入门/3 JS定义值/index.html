常量

  • 任何一个具体的数据都是常量，例如12是个常量
  • 和变量累死，我们设置一个常量（也就是一个名字），给其存储一个值，但是这个存储的值不能修改

JS中命名规范

  • JS中严格区分大小写
  • 遵循国际命名规则”驼峰命名法”

    第一个单词首字母小写，其余每一个有意义单词首字母大写

  • 命名的时候可以使用$、_、数字、字母，但是数字不能作为名字的第一位
  • JS中很多字都是有特殊含义的，我们这些词，叫做关键词；现在没有特殊含义，以后可能会作为关键的，我们叫做保留字；二关键词和保留字不可以随便用来命名；
